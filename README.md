# Analizador de Productos Químicos (Cancerígenos)

Este proyecto es una herramienta de la **Dirección Técnica IA LAB** diseñada para identificar y evaluar riesgos asociados a agentes químicos cancerígenos, mutágenos y reprotóxicos.

## Funcionalidad Principal

- **Asistente (Wizard)**: Guiado paso a paso para la evaluación de riesgos.
- **Identificación**: Detección de agentes CMR.
- **Evaluación**: Análisis según normativa RD 665/1997.

## Configuración

El nombre interno del proyecto es `direccion-tecnica-chemicals`.
